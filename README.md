# Testing WEB system

# Software you need

- Node.js

# Install dependecies

```
npm i
```

# Run tests

```
npm test
```

# Run system

```
npm run dev
```

Open on browser:

- http://localhost:3000 - lecturer view
- http://localhost:3000/test/:id - student view
- http://localhost:3000/api/tests/:id/results - results
